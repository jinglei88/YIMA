# 易码

易码是一门中文编程语言，目标是让初学者直接用中文写出可运行程序。  
本项目当前处于可用但持续演进阶段，文档只描述已实现行为。

## 环境要求

- Python 3.9+
- 源码文件统一使用 UTF-8 编码

## 快速运行

```bash
# 交互模式（适合单行测试）
python 易码.py

# 运行脚本文件
python 易码.py 示例/欢迎.ym

# 图形化编辑器
python 易码编辑器.py
```

## 已实现语法（核心）

- 输出与输入：`显示`、`输入`
- 函数：`功能`、`返回`
- 条件：`如果 ... 就`、`否则如果`、`不然`
- 循环：`当 ... 的时候`、`重复 ... 次`、`遍历 ... 里的每一个 叫做 ...`
- 循环控制：`停下`、`略过`
- 异常：`尝试`、`如果出错`、`叫做`
- 模块：`引入`、`用 ... 中的 ...`
- 面向对象：`定义图纸`、`造一个`、`它的`

## 运算符（真实可用）

- 算术：`+ - * / % ** //`
- 比较：`> < >= <= == !=`
- 逻辑：`并且`、`而且`、`或者`、`取反`、`!`

说明：中文算术/比较词（例如“加上”“大于”）目前不是官方语法入口，请使用符号运算。

## 最小示例

```ym
姓名 = 输入 "请输入姓名："
如果 姓名 == 空 就
    姓名 = "匿名用户"
显示 "你好，【姓名】"
```

```ym
功能 求和(a, b)
    返回 a + b

显示 求和(3, 4)
```

```ym
水果 = ["苹果", "香蕉", "橘子"]
遍历 水果 里的每一个 叫做 当前
    显示 当前
```

## 模块导入（当前行为）

- `引入 "xxx"`：导入 `.ym` 文件或 Python 模块
- `引入 "xxx" 叫做 别名`：以别名访问
- `用 "xxx.ym" 中的 名称`：从易码模块精确引入
- `用 "math" 中的 sqrt`：从 Python 模块精确引入

`.ym` 模块默认搜索路径：当前工作目录、`示例/` 目录。

性能说明：同一运行会话中，`.ym` 模块按“绝对路径 + 文件修改时间”缓存，重复引入不会重复执行模块顶层代码。

## 运行时开关

- `YIMA_STRICT_SCOPE=1`：开启严格局部作用域（函数内赋值默认写入当前局部，不向上覆盖父作用域同名变量）。
- 默认不开启，保持当前示例兼容行为。

Windows PowerShell 示例：

```powershell
$env:YIMA_STRICT_SCOPE="1"
python 易码.py 示例\M11深度测试.ym
Remove-Item Env:YIMA_STRICT_SCOPE
```
