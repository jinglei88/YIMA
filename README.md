## ✨ 为什么有易码？

所有所谓的"中文编程语言"都只是翻译，`=`、`>=`、缩进报错……思维还是外国人的。

**易码不翻译，易码是原创。** 完全用中国人说话的方式来写代码。

```
说 "你好，世界！"
```
```
让 名字 = 问 "你叫什么名字？"
说 "很高兴认识你，【名字】！"
```

就是这么直白。

## 📖 语法速成（5 分钟学会）

### 1. 说话（输出）
```
说 "你好呀！"
```

### 2. 记东西（变量）
```
让 名字 = "张三"
让 年龄 = 18
说 "我叫【名字】，今年【年龄】岁。"
```

### 3. 问问题（输入）
```
让 回答 = 问 "你今年几岁了？"
```

### 4. 算数
```
让 总价 = 5 * 4
让 剩余 = 100 减去 25
```
> 支持大白话 `加上/减去/乘以/除以` 和数学符号 `+ - * /`

### 5. 选择（条件）
```
如果 年龄 大于 18
    说 "成年了！"
不然
    说 "还是小孩！"
结束
```

### 6. 循环
```
重复 3 次
    说 "重要的事情说三遍！"
结束
```

### 7. 列表
```
让 爱好 = ["编程", "画画", "游戏"]
说 爱好[0]
加入(爱好, "音乐")
说 长度(爱好)
```

### 8. 自定义功能（函数）
```
功能 打招呼(名字)
    说 "你好呀，【名字】！"
结束

打招呼("小明")
```

### 9. 引入 Python 库（无限扩展）
```
引入 "math" 叫做 数学库
让 结果 = 数学库.sqrt(144)
说 "144的平方根是：【结果】"
```
> 一行代码，整个 Python 生态尽在手中。

## ⚠️ 词句对照表

| 传统概念 | 易码（大白话） |
|----------|----------------|
| `x = value` | `让 x = value` |
| `print(x)` | `说 x` |
| `input()` | `问 "提示"` |
| `if ... :` | `如果 ...` |
| `elif ... :` | `否则如果 ...` |
| `else:` | `不然` |
| `while ... :` | `当 ... 的时候` |
| `for i in range(n):` | `重复 n 次` |
| `for x in list:` | `取出 列表 里的每一个 叫做 x` |
| `def f(x):` | `功能 f(x)` |
| `return` | `交出` |
| `import math` | `引入 "math"` |
| `}` / 缩进 | `结束` |
| `[1,2,3]` | `[1, 2, 3]` |
| `list[0]` | `列表[0]` |
| `True` / `False` | `对` / `错` |
| `None` | `无` |

## 📁 内置工具函数

| 函数 | 用途 |
|------|------|
| `排列(元素...)` | 创建列表 |
| `加入(列表, 元素)` | 列表末尾添加 |
| `长度(对象)` | 获取长度 |

| `删除(列表, 序号)` | 删除指定位置元素 |

| `变文字(数字)` | 数字转文字 |
| `取随机数(最小, 最大)` | 随机整数 |

## 🎮 示例：猜数字游戏（GUI 版）

```
让 秘密数字 = 取随机数(1, 100)
让 猜了几次 = 0

让 窗口 = 建窗口("猜数字小游戏", 350, 300)
让 提示 = 加文字(窗口, "猜一个 1~100 的数字")
让 输入框 = 加输入框(窗口)

功能 猜一猜()
    让 玩家数字 = 变数字(读输入(输入框))
    让 猜了几次 = 猜了几次 + 1
    如果 玩家数字 == 秘密数字
        弹窗("恭喜", "你猜对了！")
    不然
        如果 玩家数字 > 秘密数字
            改文字(提示, "太大了！")
        不然
            改文字(提示, "太小了！")
        结束
    结束
结束

加按钮(窗口, "确认", "猜一猜")
打开界面(窗口)
```

## 🎯 设计原则

1. **不用符号**：告别 `{}`、`:`、`>=`、`!=`
2. **不靠缩进**：`结束` 收尾，缩进只为美观
3. **符合中文语序**：`让 名字 = 问 "你叫什么？"`
4. **无限扩展**：`引入` 一键接入 Python 万亿级生态
