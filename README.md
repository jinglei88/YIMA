# 易码

易码是一门中文编程语言，目标是让初学者直接用中文写出可运行程序。  
当前进入 `v1.0` 稳定期：核心语法冻结，能力以增量扩展为主。

## 环境要求

- Python 3.9+
- 源码文件统一使用 UTF-8 编码

## 快速运行

```bash
# 交互模式（适合单行测试）
python 易码.py

# 运行脚本文件
python 易码.py 示例/欢迎.ym

# 图形化编辑器
python 易码编辑器.py

# 查看命令行帮助
python 易码.py --help

# 直接执行一段源码
python 易码.py -c "显示 1 + 1"

# 显示版本号
python 易码.py --version
```

常用 CLI 参数：

- `-c/--code`：直接执行一段易码源码。
- `--strict-scope`：强制开启严格局部作用域（覆盖环境变量）。
- `--no-strict-scope`：强制关闭严格局部作用域（覆盖环境变量）。
- `--traceback`：内部异常时显示 Python 调用栈。
- `-v/--version`：输出版本号并退出。

CLI 退出码约定：

- `0`：成功。
- `2`：命令行参数错误（`argparse`）。
- `3`：文件读取/路径错误。
- `10`：词法错误。
- `11`：语法错误。
- `12`：运行错误。
- `20`：解释器内部错误。

## 已实现语法（核心）

- 输出与输入：`显示`、`输入`
- 功能：`功能`、`返回`
- 条件：`如果 ... 就`、`否则如果`、`不然`
- 循环：`当 ... 的时候`、`重复 ... 次`、`遍历 ... 里的每一个 叫做 ...`
- 循环控制：`停下`、`略过`
- 异常：`尝试`、`如果出错`、`叫做`
- 模块：`引入 "模块" 叫做 别名`（统一写法）
- 面向对象：`定义图纸`、`造一个`、`它的`

## 运算符（真实可用）

- 算术：`+ - * / % ** //`
- 比较：`> < >= <= == !=`
- 逻辑：`并且`、`而且`、`或者`、`取反`、`!`

说明：中文算术/比较词（例如“加上”“大于”）目前不是官方语法入口，请使用符号运算。

## v1.0 冻结策略

- 核心语法：`如果/否则如果/不然`、`功能`、`定义图纸`、`引入 ... 叫做 ...` 不做破坏性变更。
- 内置能力：现有名称不重命名、不删除（按兼容策略新增）。
- 历史同义词不再新增，文档默认只展示官方写法。

详细规则见：`文档/v1.0冻结与回归策略.md`

## 最小示例

```ym
姓名 = 输入 "请输入姓名："
如果 姓名 == 空 就
    姓名 = "匿名用户"
显示 "你好，【姓名】"
```

模板字符串中如果要字面量显示 `【】`，可用以下任一写法：

```ym
显示 "按钮：【【继续探索】】"
显示 "按钮：\【继续探索\】"
```

```ym
功能 求和(a, b)
    返回 a + b

显示 求和(3, 4)
```

```ym
水果 = ["苹果", "香蕉", "橘子"]
遍历 水果 里的每一个 叫做 当前
    显示 当前
```

## 模块导入（当前行为）

- 官方写法：`引入 "模块名" 叫做 名称`
- 如果模块里存在同名导出（或 Python 模块同名属性），会直接绑定该名称。
- 如果不存在同名导出，则该名称绑定整个模块对象。

示例：

- `引入 "math" 叫做 数学库`，然后 `数学库.sqrt(16)`
- `引入 "math" 叫做 pi`，可直接 `显示 pi`
- `引入 "数据层.ym" 叫做 数据`，然后 `数据.查询()`

`.ym` 模块默认搜索路径：当前工作目录、`示例/` 目录。

性能说明：同一运行会话中，`.ym` 模块按“绝对路径 + 文件修改时间”缓存，重复引入不会重复执行模块顶层代码。

## 运行时开关

- `YIMA_STRICT_SCOPE=1`：开启严格局部作用域（函数内赋值默认写入当前局部，不向上覆盖父作用域同名变量）。
- `YIMA_GUI_THEME=深色`：GUI 使用深色主题（同义值：`dark`、`专业深色`；默认是简洁浅色）。
- 默认不开启，保持当前示例兼容行为。
- 命令行可用 `--strict-scope` / `--no-strict-scope` 临时覆盖该环境变量。

Windows PowerShell 示例：

```powershell
$env:YIMA_STRICT_SCOPE="1"
python 易码.py 示例\M11深度测试.ym
Remove-Item Env:YIMA_STRICT_SCOPE
```
