# 易码语言规范（以当前实现为准）

本规范只描述当前解释器已经实现并通过示例验证的行为。  
阅读顺序建议：

- 先看第 1 节（入门推荐写法）
- 再看第 3 节（完整关键字表）

版本说明：`v1.0` 核心语法冻结，具体策略见 `文档/v1.0冻结与回归策略.md`。

## 1. 入门推荐写法（先学这个）

目标：同一种能力只记一种写法，先把程序跑起来。

### 1.1 官方推荐句式

| 场景 | 官方推荐写法 | 示例 |
|---|---|---|
| 赋值 | `变量 = 表达式` | `名字 = "小明"` |
| 输出 | `显示 表达式` | `显示 名字` |
| 输入 | `变量 = 输入 提示表达式` | `名字 = 输入 "请输入名字："` |
| 条件 | `如果 条件 就` | `如果 分数 >= 60 就` |
| 否则如果 | `否则如果 条件 就` | `否则如果 分数 >= 80 就` |
| 否则 | `不然` | `不然` |
| 当循环 | `当 条件 的时候` | `当 i < 10 的时候` |
| 固定次数循环 | `重复 次数表达式 次 叫做 序号` | `重复 5 次 叫做 序号` |
| 遍历循环 | `遍历 集合 里的每一个 叫做 元素` | `遍历 列表 里的每一个 叫做 项` |
| 函数定义 | `功能 名称(参数)` | `功能 求和(a, b)` |
| 返回 | `返回` 或 `返回 表达式` | `返回 a + b` |
| 异常处理 | `尝试 ... 如果出错 叫做 错误` | 见第 7.6 节 |

### 1.2 官方风格约定

1. 函数和图纸参数，推荐统一使用括号写法：

```ym
功能 面积(长, 宽)
定义图纸 玩家(名字, 血量)
```

2. 条件分支推荐总是写 `就`，减少歧义。
3. 运算统一使用符号（`+ - * /` 等），不要混用中文运算词。
4. `尝试` 推荐总是配 `如果出错`，避免漏错。

## 2. 文件与缩进

1. 源码建议使用 UTF-8 编码。
2. 代码块由缩进决定，不使用块结束关键字。
3. 缩进可用空格或 Tab（Tab 按 4 个空格处理），同一层级必须对齐。
4. 注释支持两种写法：

```ym
# 这是注释
标注：这也是注释
```

## 3. 关键字全表（官方入口）

### 3.1 核心关键字

| 关键字 | 作用 | 使用方式 | 示例 |
|---|---|---|---|
| `显示` | 输出表达式值 | `显示 表达式` | `显示 "你好"` |
| `输入` | 读取用户输入（返回文本） | `变量 = 输入 提示表达式` | `名字 = 输入 "请输入名字："` |
| `功能` | 定义函数 | `功能 名称(参数)` | `功能 求和(a, b)` |
| `返回` | 从函数返回值 | `返回` 或 `返回 表达式` | `返回 a + b` |
| `引入` | 引入模块并注册名称 | `引入 "模块" 叫做 别名` | `引入 "math" 叫做 数学库` |
| `叫做` | 给对象命名（别名/变量名） | 用于 `引入`、`重复`、`遍历`、`如果出错` | `重复 5 次 叫做 序号` |

### 3.2 条件与异常关键字

| 关键字 | 作用 | 使用方式 | 示例 |
|---|---|---|---|
| `如果` | 条件分支起点 | `如果 条件 就` | `如果 分数 >= 60 就` |
| `就` | 条件后连接词 | 常跟在 `如果`/`否则如果` 后 | `如果 条件 就` |
| `否则如果` | 次级条件分支 | `否则如果 条件 就` | `否则如果 分数 >= 80 就` |
| `不然` | 兜底分支 | `不然` | `不然\n    显示 "未通过"` |
| `尝试` | 异常保护块开始 | `尝试` | `尝试\n    内容 = 读文件("a.txt")` |
| `如果出错` | 异常捕获分支 | `如果出错` / `如果出错 叫做 变量` | `如果出错 叫做 错误信息` |

### 3.3 循环关键字

| 关键字 | 作用 | 使用方式 | 示例 |
|---|---|---|---|
| `当` | 条件循环起点 | `当 条件 的时候` | `当 i < 10 的时候` |
| `的时候` | `当` 语句结束标记 | 必须出现在 `当 条件` 后 | `当 x > 0 的时候` |
| `重复` | 固定次数循环起点 | `重复 次数表达式 次` | `重复 5 次` |
| `次` | `重复` 语句计数单位 | 必须出现在次数后 | `重复 n 次` |
| `遍历` | 遍历可迭代对象 | `遍历 集合 里的每一个 叫做 元素` | `遍历 列表 里的每一个 叫做 项` |
| `里的每一个` | 遍历结构标记 | 必须出现在 `遍历 集合` 后 | `遍历 水果 里的每一个 叫做 当前` |
| `停下` | 跳出当前循环 | 单独成句 | `停下` |
| `略过` | 进入下一轮循环 | 单独成句 | `略过` |

### 3.4 面向对象关键字

| 关键字 | 作用 | 使用方式 | 示例 |
|---|---|---|---|
| `定义图纸` | 定义类型（类） | `定义图纸 名称(参数)` | `定义图纸 玩家(名字, 血量)` |
| `造一个` | 创建实例 | `变量 = 造一个 图纸名(参数)` | `角色 = 造一个 玩家("小白", 100)` |
| `它的` | 访问/设置实例属性 | `它的 属性` 或 `它的 属性 = 表达式` | `它的 血量 = 它的 血量 - 10` |

## 4. 保留字面量词

这些词不是语句关键字，但在词法层按固定含义处理：

- `对`：布尔真
- `错`：布尔假
- `空`：空值

## 5. 字面量与数据

- 数字：`1`、`3.14`
- 文本：`"你好"`、`'你好'`、`“你好”`、`‘你好’`
- 布尔：`对`、`错`
- 空值：`空`
- 列表：`[1, 2, 3]`
- 字典：`{"姓名": "小明", "年龄": 18}`
- 模板字符串：`"你好，【名字】"`
  - 字面量 `【】` 写法：`"【【继续探索】】"` 或 `"\\【继续探索\\】"`

## 6. 表达式与运算符

### 6.1 运算符（当前官方入口）

- 算术：`+ - * / % ** //`
- 比较：`> < >= <= == !=`
- 逻辑：`并且`、`而且`、`或者`、`取反`、`!`

### 6.2 当前不作为语法入口的写法

以下中文运算词当前不会被当成运算符解析：

- `加上` `减去` `乘以` `除以`
- `大于` `小于` `等于` `不等于` `至少是` `最多是`

请使用符号运算写法。

## 7. 语句示例

### 7.1 赋值

```ym
名字 = "小明"
年龄 = 18
```

### 7.2 输出与输入

```ym
名字 = 输入 "请输入名字："
显示 名字
```

### 7.3 条件分支

```ym
如果 分数 >= 90 就
    显示 "优秀"
否则如果 分数 >= 60 就
    显示 "及格"
不然
    显示 "需要加强"
```

### 7.4 循环

```ym
当 计数 < 3 的时候
    显示 计数
    计数 += 1
```

```ym
重复 5 次 叫做 序号
    显示 序号
```

```ym
遍历 水果 里的每一个 叫做 当前
    显示 当前
```

### 7.5 函数

```ym
功能 加法(a, b)
    返回 a + b

功能 面积(长, 宽)
    返回 长 * 宽
```

### 7.6 异常处理

```ym
尝试
    内容 = 读文件("不存在.txt")
如果出错 叫做 错误信息
    显示 错误信息
```

补充说明：

- `如果出错` 可省略变量名；
- 若 `尝试` 后未写 `如果出错` 分支，异常会继续向外抛出，不会被静默吞掉；
- 错误变量写法已统一：如需变量请写 `如果出错 叫做 错误`。

### 7.7 模块

```ym
引入 "math" 叫做 数学库
显示 数学库.sqrt(144)

引入 "math" 叫做 pi
显示 pi
```

### 7.8 面向对象

```ym
定义图纸 玩家(名字, 血量)
    它的 名字 = 名字
    它的 血量 = 血量

    功能 受伤(伤害)
        它的 血量 = 它的 血量 - 伤害

角色 = 造一个 玩家("小白", 100)
角色.受伤(20)
显示 角色.血量
```

### 8.10 已知使用约束（必须注意）

1. `绑定按键` 的第一个参数必须是功能对象（如 `绑定按键(向左转, "Left")`），不要传字符串。
2. `打开界面` 与 `定格` 会阻塞当前脚本执行，自动化测试应单独拆分。
3. GUI/画板相关函数在无图形环境中可能不可用，建议做降级或跳过策略。

## 9. 运行时开关与行为约束

1. 当前版本不维护历史同义词兼容层；请使用官方写法（例如使用 `新列表`，不要写 `排列` / `是一份清单`）。

2. 严格局部作用域（可选）：
- 默认行为：函数内对同名变量赋值会向上更新外层作用域（保持历史示例兼容）。
- 可选开关：设置环境变量 `YIMA_STRICT_SCOPE=1` 后，函数内赋值默认写入局部作用域，不向上覆盖父作用域变量。

## 10. 错误输出格式

当前版本错误信息统一包含三部分：

1. 位置：行号/列号（如可用）
2. 原因：发生错误的直接原因
3. 建议：可执行的修复方向

示例：

```text
语法错误（第 3 行，第 5 列）：
  原因：此处应为【右括号】，实际读取到【换行】。
  建议：请检查当前位置前后是否缺少【右括号】，或多写了其它符号。
```
