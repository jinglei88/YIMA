# 易码语言规范 v4.0（大白话版）

## 一、设计哲学

易码是一门**纯原创的中文编程语言**，完全根据中国人的自然语言思维设计。**每个功能只有一种写法**，零基础也能看懂。

---

## 二、关键字速查表

### 2.1 核心动作
| 词汇 | 用途 | 示例 |
|------|------|------|
| `显示` | 输出内容 | `显示 "你好"` |
| `问` | 获取用户输入 | `回答 = 问 "你叫什么？"` |
| `功能` | 定义可复用的动作 | `功能 打招呼(名字)` |
| `交出` | 返回结果 | `交出 结果` |
| `引入` | 导入库 | `引入 "math" 叫做 数学库` |
| `用...中的...` | 精确引入 | `用 "工具.ym" 中的 求平方` |

### 2.2 逻辑控制
| 词汇 | 用途 | 示例 |
|------|------|------|
| `如果...就` | 条件判断 | `如果 年龄 > 18 就` |
| `否则如果` | 多分支 | `否则如果 年龄 == 18` |
| `不然` | 否则 | `不然` |
| `完事` | 代码块结尾 | `完事` |

### 2.3 循环
| 词汇 | 用途 | 示例 |
|------|------|------|
| `重复 N 次` | 固定次数 | `重复 3 次` |
| `当...的时候` | 条件循环 | `当 计数 < 10 的时候` |
| `取出...里的每一个 叫做` | 遍历 | `取出 列表 里的每一个 叫做 项` |
| `停下` | 跳出循环 | `停下` |
| `略过` | 跳过本次 | `略过` |

### 2.4 比较与逻辑
| 符号 | 含义 |
|------|------|
| `==` | 等于 |
| `!=` | 不等于 |
| `>` | 大于 |
| `<` | 小于 |
| `>=` | 大于等于 |
| `<=` | 小于等于 |
| `而且` | 并且 |
| `或者` | 或者 |
| `取反` | 取反 |

### 2.5 运算符
统一使用数学符号：`+` `-` `*` `/` `%` `**` `//`

### 2.6 数据值
| 词汇 | 含义 |
|------|------|
| `对` | True |
| `错` | False |
| `空` | None |

## 三、句式语法

### 3.1 赋值
```
玩家血量 = 100
名字 = "林克"
```

### 3.2 输出与输入
```
显示 "欢迎来到游戏！"
回答 = 问 "你叫什么名字？"
```

**模板字符串**（用 `【变量名】` 在文本中嵌入变量）：
```
名字 = "小明"
显示 "你好呀，【名字】！欢迎回来。"
```

### 3.3 条件判断
```
如果 成绩 >= 90 就
    显示 "太优秀了！"
否则如果 成绩 >= 60 就
    显示 "及格了！"
不然
    显示 "要加油了！"
完事
```

### 3.4 循环

**条件循环：**
```
计步器 = 0
当 计步器 < 10 的时候
    显示 计步器
    计步器 += 1
完事
```

**遍历列表：**
```
水果 = ["苹果", "香蕉", "橘子"]
取出 水果 里的每一个 叫做 当前水果
    显示 "我喜欢吃" + 当前水果
完事
```

**固定次数：**
```
重复 5 次
    显示 "加油！"
完事
```

### 3.5 异常捕获
```
尝试
    内容 = 读文件("不存在的文件.txt")
    显示 内容
如果出错 叫做 错误信息
    显示 "读取失败：" + 错误信息
完事
```

### 3.6 功能定义（函数）
```
功能 打招呼(名字)
    显示 "你好呀，【名字】！"
完事

打招呼("小明")
```

带返回值：
```
功能 算面积(长, 宽)
    交出 长 * 宽
完事

面积 = 算面积(5, 4)
显示 "面积是：【面积】"
```

### 3.7 引入库（FFI 外挂系统）
```
引入 "math" 叫做 数学库
结果 = 数学库.sin(3.14159 / 2)
显示 "sin(90°) = 【结果】"
```

## 四、数据结构

### 4.1 数据类型
- **整数**：`123`，`-5`
- **小数**：`3.14`
- **文本**：`"你好"`
- **布尔值**：`对`，`错`
- **列表**：`[1, 2, 3]`
- **字典**：`{"名字": "小明", "年龄": 18}`

### 4.2 列表操作
```
购物车 = ["苹果", "牛奶"]
加入(购物车, "面包")
显示 长度(购物车)
显示 购物车[0]
删除(购物车, 0)
```

### 4.3 内置工具函数

| 函数 | 用途 |
|------|------|
| `加入(列表, 元素)` | 往列表末尾添加元素 |
| `长度(对象)` | 获取列表、字典或文本长度 |
| `删除(列表, 序号)` | 删除指定位置的元素 |
| `转数字(文本)` | 文本转数字 |
| `转文字(数字)` | 数字转文字 |
| `取随机数(最小, 最大)` | 生成随机整数 |

### 4.4 字典操作

```
资料 = {"名字": "张三", "年龄": 18}
显示 资料["名字"]
显示 资料.名字

资料.年龄 = 19
资料.成绩 = 100
```

| 函数 | 用途 |
|------|------|
| `所有键(字典)` | 获取字典的所有键名列表 |
| `所有值(字典)` | 获取字典的所有值列表 |
| `有没有(字典, 键)` | 判断某个键是否在字典中 |

### 4.5 文本与文件操作

| 函数 | 用途 |
|------|------|
| `截取(文本, 起, 止)` | 截取字符串 |
| `查找(文本, 子串)` | 查找子串位置 |
| `替换(文本, 旧, 新)` | 替换文本 |
| `分割(文本, 分隔符)` | 拆成列表 |
| `去空格(文本)` | 去掉首尾空白 |
| `包含(文本, 子串)` | 判断是否包含子串 |
| `读文件(路径)` | 读取文件所有文本 |
| `写文件(路径, 内容)` | 将内容写入文件 |
| `追加文件(路径, 内容)` | 追加到文件末尾 |

### 4.6 多文件开发

```
# 工具.ym
功能 算总价(单价)
    交出 单价 + 100
完事
```

引入方式：
```
# 主代码.ym
引入 "工具.ym" 叫做 我的工具箱
结果 = 我的工具箱.算总价(5)

# 或者精确引入
用 "工具.ym" 中的 算总价
结果 = 算总价(5)
```

## 五、面向对象（制造图纸）

```
制造图纸 怪物(名字, 血量, 攻击力)
    它的 名字 = 名字
    它的 血量 = 血量
    它的 攻击力 = 攻击力

    功能 挨打(伤害)
        它的 血量 = 它的 血量 - 伤害
        如果 它的 血量 <= 0 就
            显示 它的 名字 + " 被打倒了！"
        不然
            显示 它的 名字 + " 剩余血量：" + 它的 血量
        完事
    完事
完事

史莱姆 = 制造 怪物("史莱姆", 50, 5)
显示 史莱姆.名字
史莱姆.挨打(30)
```

## 八、编写规则

1. **一行一句**：不需要分号。
2. **缩进不强制**：`完事` 才是代码块的界定标志。缩进只是为了美观。
3. **注释**：用 `#` 开头。
