# 易码语言规范 v2.0（大白话版）

## 一、设计哲学

易码是一门**纯原创的中文编程语言**，完全根据中国人的自然语言思维设计。不是"把英文翻译成中文"，而是从零开始构建的大白话语法体系。

---

## 二、关键字速查表

### 2.1 核心动作
| 词汇 | 用途 | 示例 |
|------|------|------|
| `说` / `显示` | 输出内容 | `显示 "你好"` |
| `让` (已弃用) | 极简赋值 | `名字 = "张三"` |
| `把...设定为` | 经典赋值 | `把 年龄 设定为 18` |
| `问` / `询问` | 获取用户输入 | `让 回答 = 问 "你好吗？"` |
| `功能` | 定义可复用的动作 | `功能 打招呼(名字)` |
| `交出` | 返回结果 | `交出 结果` |
| `引入` | 导入 Python 库 | `引入 "math" 叫做 数学库` |

### 2.2 逻辑控制
| 词汇 | 用途 | 示例 |
|------|------|------|
| `如果...的话` | 条件判断 | `如果 年龄 > 18 的话` |
| `不然` / `不然的话` | 否则 | `不然 显示 "还没成年"` |
| `否则如果` | 多分支 | `否则如果 年龄 == 18` |
| `结束` / `完事` | 代码块收尾 | `完事` |

### 2.3 循环
| 词汇 | 用途 | 示例 |
|------|------|------|
| `重复 N 次` | 固定次数 | `重复 3 次` |
| `当...的时候` | 条件循环 | `当 计数 < 10 的时候` |
| `只要` | 条件循环简写 | `只要 计数 < 10` |
| `取出...里的每一个 叫做` | 遍历 | `取出 列表 里的每一个 叫做 项` |
| `停下` | 跳出循环 | `停下` |
| `略过` | 跳过本次 | `略过` |

### 2.4 比较与逻辑词汇
| 大白话 | 符号等价 |
|--------|----------|
| `等于` / `是` | `==` |
| `不等于` / `不是` | `!=` |
| `大于` | `>` |
| `小于` | `<` |
| `至少是` | `>=` |
| `最多是` | `<=` |
| `并且` / `而且` | `and` |
| `或者` | `or` |
| `取反` / `反过来说` | `not` |

### 2.5 运算词汇
| 大白话 | 符号等价 |
|--------|----------|
| `加上` | `+` |
| `减去` | `-` |
| `乘以` | `*` |
| `除以` | `/` |
| `取余` | `%` |
| `拼接` | 字符串连接 |

> 注：易码同时支持传统数学符号 `+ - * / % > < == !=`，解析器自动将它们与中文词汇同等对待。

### 2.6 数据值
| 词汇 | 含义 |
|------|------|
| `对` | True |
| `错` | False |
| `无` | None |
| `空的` | 空字符串 `""` |

## 三、句式语法

### 3.1 赋值

**极简语法（推荐）：**
```
玩家血量 = 100
名字 = "林克"
```

**经典语法：**
```
把 玩家血量 设定为 100
```

### 3.2 输出与输入

```
显示 "欢迎来到游戏！"
回答 = 问 "你叫什么名字？"
```

**模板字符串**（用 `【变量名】` 在文本中嵌入变量）：
```
名字 = "小明"
显示 "你好呀，【名字】！欢迎回来。"
```

### 3.3 条件判断

```
如果 成绩 >= 90
    显示 "太优秀了！"
否则如果 成绩 >= 60
    显示 "及格了！"
不然
    显示 "要加油了！"
完事
```

> `的话` 是可选的修饰词，写不写都行。

### 3.4 循环

**固定次数：**
```
当 计步器 < 10 的时候
    如果 计步器 == 5
        略过
    完事
    如果 计步器 == 8
        停下
    完事
    显示 计步器
    把 计步器 设定为 计步器 + 1
完事
```

### 3.4 异常捕获 (尝试...如果出错...结束)
当程序可能因为某些原因报错（比如去读一个不存在的文件），可以用 `尝试` 结构防止程序崩溃。

```
尝试
    内容 = 读文件("乱写的路径.txt")
    显示 内容
如果出错 叫做 错误信息
    显示 "哎呀，读取失败了，原因是：" + 错误信息
完事
```
*   `尝试` 到 `如果出错` 之间的代码会被保护起来。
*   一旦报错，程序不会立刻退出，而是跳到 `如果出错` 后面执行。
*   `叫做 错误信息` 是可选的，用来获取具体的报错文字。也可以只写 `如果出错` 然后直接默默处理。

**遍历列表：**
```
水果 = ["苹果", "香蕉", "橘子"]
取出 水果 里的每一个 叫做 当前水果
    显示 "我喜欢吃" + 当前水果
完事
```

### 3.5 功能定义（函数）

```
功能 打招呼(名字)
    显示 "你好呀，【名字】！"
完事

打招呼("小明")
```

带返回值：
```
功能 算面积(长, 宽)
    交出 长 * 宽
完事

面积 = 算面积(5, 4)
显示 "面积是：【面积】"
```

### 3.6 引入 Python 库（FFI 外挂系统）

易码可以无缝调用 Python 生态中的任意库：

```
引入 "math" 叫做 数学库
结果 = 数学库.sin(3.14159 / 2)
显示 "sin(90°) = 【结果】"

引入 "time"
time.sleep(1)
```

## 四、数据结构

### 4.1 数据类型
- **整数**：`123`，`-5`
- **小数**：`3.14`
- **文本**：`"你好"`
- **布尔值**：`对`，`错`
- **列表**：`[1, 2, 3]` 或 `["苹果", "香蕉"]`
- **字典**：`{"名字": "小明", "年龄": 18}`

### 4.2 列表操作

```
购物车 = ["苹果", "牛奶"]
加入(购物车, "面包")       # 添加元素
显示 长度(购物车)             # 输出 3
显示 购物车[0]               # 输出 "苹果"
显示 提取(购物车, 1)          # 输出 "牛奶"
删除(购物车, 0)             # 删除第一个
```

也可以用内置函数创建列表：
```
数字们 = 新列表(10, 20, 30)
```

### 4.3 基础内置工具

| 函数 | 用途 |
|------|------|
| `新列表(元素...)` | 创建列表 |
| `加入(列表, 元素)` | 往列表末尾添加元素 |
| `长度(对象)` | 获取列表、字典或文本长度 |
| `提取(列表, 序号)` | 取出指定位置的元素 |
| `删除(列表, 序号)` | 删除指定位置的元素 |
| `转数字(文本)` | 文本转数字 |
| `转文字(数字)` | 数字转文字 |
| `取随机数(最小, 最大)` | 生成随机整数 |

### 4.4 字典操作

```
资料 = {"名字": "张三", "年龄": 18}
显示 资料["名字"]          # 输出 张三
显示 资料["年龄"]          # 输出 18
资料["成绩"] = 100       # 不支持这种写法，如果需要更改，请重新生成字典或使用 Python 原生方法。
```

| 函数 | 用途 |
|------|------|
| `所有键(字典)` | 获取字典的所有键名列表 |
| `所有值(字典)` | 获取字典的所有值列表 |
| `有没有(字典, 键)` | 判断某个键是否在字典中，返回对/错 |

### 4.5 文本与文件操作

| 函数 | 用途 |
|------|------|
| `截取(文本, 起, 止)` | 截取字符串（比如截取 0 到 2） |
| `查找(文本, 子串)` | 查找子串位置，找不到返回 -1 |
| `替换(文本, 旧, 新)` | 将文本中的旧字符串换成新字符串 |
| `分割(文本, 分隔符)` | 按照分隔符把文本拆成列表 |
| `去空格(文本)` | 去掉文本开头和结尾的空白 |
| `包含(文本, 子串)` | 判断文本中是否包含子串，返回对/错 |
| `读文件(路径)` | 读取并返回文件所有文本 |
| `写文件(路径, 内容)` | 将内容写入文件（覆盖） |
| `追加文件(路径, 内容)`| 将内容加到文件末尾 |

### 4.6 多文件开发 (模块化)

当你写了很长的代码时，可以拆分成多个文件。
假设你写了一个叫做 `工具.ym` 的文件，里面定义了功能：
```
# 工具.ym
功能 算总价 需要 单价
    交出 单价 + 100
完事
寄语 = "这是一本好书"
```

你可以建立另一个代码文件引入它：
```
# 主代码.ym
引入 "工具.ym" 叫做 我的工具箱

显示 我的工具箱.寄语
结果 = 我的工具箱.算总价(5)
显示 结果
```

## 五、编写规则

1. **一行一句**：不需要分号。
2. **缩进不强制**：`结束` 才是代码块的界定标志。缩进只是为了美观。
3. **注释**：用 `#` 开头。
4. **兼容性**：`教电脑` 和 `定义` 也可以用来定义功能（向后兼容）。`学完了` 也是合法的代码块结尾。
