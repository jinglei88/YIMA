# 易码开发指南（当前实现）

本文件保留为快速入口。完整开发教程请看：`文档/开发者教程.md`。

## 1. 环境要求

- Python 3.9+
- UTF-8 终端与文件编码

## 2. 本地运行

```bash
python 易码.py
python 易码.py 示例/欢迎.ym
python 易码编辑器.py
```

## 3. 快速回归命令

```bash
python -m py_compile 易码.py yima\词法分析.py yima\语法分析.py yima\语法树.py yima\解释器.py yima\信号.py 易码编辑器.py
python 易码.py 示例\M10综合测试.ym
python 易码.py 示例\M11模块化测试.ym
python 易码.py 示例\M12容错测试.ym
```

推荐使用一键回归脚本（包含以上检查并额外覆盖除零、严格作用域、模块缓存语义）：

```bash
python scripts/run_regression.py
```

仅验证错误体验（20 个高频错误场景）：

```bash
python scripts/run_error_regression.py
```

## 4. 开发流程（简版）

1. 改 `yima/词法分析.py`。
2. 改 `yima/语法分析.py` 和 `yima/语法树.py`。
3. 改 `yima/解释器.py`。
4. 补文档与示例。
5. 跑回归命令。
