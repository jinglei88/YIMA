# 易码官方写法速查表

本页只放“推荐写法”。目标是：初学者照着写就能跑。

## 1. 先记住这 12 个句式

| 场景 | 官方推荐写法 | 示例 |
|---|---|---|
| 赋值 | `变量 = 表达式` | `名字 = "小明"` |
| 输出 | `显示 表达式` | `显示 名字` |
| 输入 | `变量 = 输入 提示` | `名字 = 输入 "请输入名字："` |
| 条件 | `如果 条件 就` | `如果 分数 >= 60 就` |
| 否则如果 | `否则如果 条件 就` | `否则如果 分数 >= 80 就` |
| 否则 | `不然` | `不然` |
| 当循环 | `当 条件 的时候` | `当 i < 10 的时候` |
| 固定次数循环 | `重复 次数 次 叫做 序号` | `重复 5 次 叫做 序号` |
| 遍历循环 | `遍历 集合 里的每一个 叫做 元素` | `遍历 列表 里的每一个 叫做 项` |
| 功能定义 | `功能 名称(参数)` | `功能 求和(a, b)` |
| 返回 | `返回 表达式` | `返回 a + b` |
| 异常处理 | `尝试 ... 如果出错 叫做 错误` | 见下方模板 |

## 2. 推荐模板

### 2.1 条件分支

```ym
如果 条件1 就
    显示 "命中条件1"
否则如果 条件2 就
    显示 "命中条件2"
不然
    显示 "兜底分支"
```

### 2.2 循环

```ym
当 计数 < 3 的时候
    显示 计数
    计数 += 1
```

```ym
重复 3 次 叫做 序号
    显示 序号
```

```ym
遍历 列表 里的每一个 叫做 当前
    显示 当前
```

### 2.3 功能

```ym
功能 加法(a, b)
    返回 a + b
```

### 2.4 异常处理

```ym
尝试
    内容 = 读文件("不存在.txt")
如果出错 叫做 错误
    显示 错误
```

### 2.5 模块

```ym
引入 "math" 叫做 数学库
显示 数学库.sqrt(144)

引入 "math" 叫做 pi
显示 pi
```

### 2.6 网格表格（GUI）

```ym
窗口 = 建窗口("表格示例", 600, 420)
表格 = 加表格(窗口, ["平台", "账号", "密码"], 10)
表格加行(表格, ["GitHub", "alice", "abc123"])
表格加行(表格, {"平台": "邮箱", "账号": "alice@qq.com", "密码": "******"})
打开界面(窗口)
```

### 2.7 模板字符串与字面量【】

```ym
名字 = "小明"
显示 "你好，【名字】"              # 模板替换
显示 "按钮：【【继续探索】】"      # 字面量【】
显示 "按钮：\【继续探索\】"        # 字面量【】
```

### 2.8 JSON + 本地数据库（SQLite）

```ym
引入 "数据工具" 叫做 数据
引入 "本地数据库" 叫做 库

配置 = {"作者": "景磊", "项目": "易码"}
数据.写JSON("配置.json", 配置)
显示 数据.读JSON("配置.json")
数据.写CSV("用户.csv", 新列表({"账号": "demo", "状态": "启用"}), 新列表("账号", "状态"))
显示 数据.读CSV("用户.csv")

连接 = 库.打开数据库("演示.db")
库.执行SQL(连接, "CREATE TABLE IF NOT EXISTS 用户(id INTEGER PRIMARY KEY, 名字 TEXT, 年龄 INTEGER)")
库.执行SQL(连接, "INSERT INTO 用户(名字, 年龄) VALUES(?, ?)", ["小明", 18])
结果 = 库.查询SQL(连接, "SELECT 名字, 年龄 FROM 用户")
显示 结果
库.关闭数据库(连接)
```

### 2.9 路径与网络请求

```ym
引入 "系统工具" 叫做 系统
引入 "网络请求" 叫做 网络

目录 = 系统.拼路径(系统.当前目录(), "数据")
系统.创建目录(目录)
显示 系统.列出目录(系统.当前目录())

响应 = 网络.发GET("https://httpbin.org/get")
如果 响应["成功"] 就
    显示 "状态码：" + 响应["状态码"]
不然
    显示 "请求失败：" + 响应["错误"]

系统.写环境变量("YIMA_ENV", "dev")
显示 系统.读环境变量("YIMA_ENV", "prod")
执行结果 = 系统.执行命令("echo YIMA_OK")
显示 执行结果["标准输出"]

功能 下载进度(已下载字节, 总字节数, 百分比)
    显示 "下载进度：" + 转文字(百分比) + "%"

下载结果 = 系统.下载文件("https://httpbin.org/bytes/256", "样例.bin", 30, 对, 下载进度, 16384)
显示 下载结果["成功"]

仅文件 = 系统.匹配文件(系统.拼路径(目录, "*"), 错, "文件")
仅目录 = 系统.匹配文件(系统.拼路径(目录, "*"), 错, "目录")
显示 仅文件
显示 仅目录
```

### 2.10 事务与 JSON 请求快捷接口

```ym
引入 "本地数据库" 叫做 库
引入 "网络请求" 叫做 网络

连接 = 库.打开数据库("订单.db", 错)  # 错=关闭自动提交，手动事务
尝试
    库.开始事务(连接)
    库.执行SQL(连接, "CREATE TABLE IF NOT EXISTS 订单(编号 TEXT, 金额 REAL)")
    库.执行SQL(连接, "INSERT INTO 订单(编号, 金额) VALUES(?, ?)", ["A001", 99.8])
    库.提交事务(连接)
如果出错 叫做 错误
    库.回滚事务(连接)
    显示 "事务失败：" + 错误
库.关闭数据库(连接)

接口结果 = 网络.发GET_JSON("https://httpbin.org/json")
如果 接口结果["成功"] 并且 接口结果["JSON"] != 空 就
    显示 "已解析 JSON"
```

### 2.11 全局/局部变量速查卡

规则（当前真实行为）：

1. 顶层赋值：全局变量。
2. `功能` 参数和功能内新建变量：局部变量。
3. 默认模式下，功能内给“外层同名变量”赋值，会向上覆盖外层值。
4. 开启 `YIMA_STRICT_SCOPE=1` 后，功能内赋值默认只写局部，不向上覆盖。

默认模式示例：

```ym
总分 = 0
功能 加分()
    总分 = 总分 + 10
加分()
显示 总分   # 10
```

严格局部模式（PowerShell）：

```powershell
$env:YIMA_STRICT_SCOPE="1"
python 易码.py 你的脚本.ym
Remove-Item Env:YIMA_STRICT_SCOPE
```

## 3. 运算符（请用符号写）

- 算术：`+ - * / % ** //`
- 比较：`> < >= <= == !=`
- 逻辑：`并且`、`而且`、`或者`、`取反`、`!`

不推荐（当前不作为官方语法入口）：

- `加上` `减去` `乘以` `除以`
- `大于` `小于` `等于` `不等于` `至少是` `最多是`

## 6. 做软件别漏的三件事

1. 先按标准结构组织：`主程序.ym + 界面/界面层.ym + 业务/业务层.ym + 数据/数据层.ym`。
2. 每次改动先跑自动回归：`python 易码.py 示例/M16全量能力测试项目/主程序.ym`。
3. 发布前补跑手动交互：
   - `python 易码.py 示例/M16全量能力测试项目/07_GUI手动交互验证.ym`
   - `python 易码.py 示例/M16全量能力测试项目/08_画板手动交互验证.ym`
4. 发版前先跑文档一致性：`python scripts/run_docs_regression.py`
5. 发版前一键跑：`python scripts/run_release_regression.py`
6. 需要补依赖/资源时，在项目根目录加 `易码打包清单.json`。

## 8. 编辑器定位问题看这里

- `[错]`：语法/错误定位
- `[提]`：一般语义提示
- 自动补全：默认自动弹出，也可手动按 `Ctrl+Space`
- 语义排序：当前作用域参数/局部变量优先
- 签名提示：可调用项会显示参数签名（如 `(a, b)`）
- 成员补全：输入 `别名.` 触发（如 `系统.`、`文件.`、`数据.`、`工具.`）
- 输入法兼容：代码区输入 `。/．/｡` 会自动转为 `.`
- 候选选择：`↑/↓` + `Enter`，或鼠标单击
- 可调用项确认后会自动补 `()` 并把光标放入括号
