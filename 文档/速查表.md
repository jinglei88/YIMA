# 易码官方写法速查表

本页只放“推荐写法”。目标是：初学者照着写就能跑。

## 1. 先记住这 12 个句式

| 场景 | 官方推荐写法 | 示例 |
|---|---|---|
| 赋值 | `变量 = 表达式` | `名字 = "小明"` |
| 输出 | `显示 表达式` | `显示 名字` |
| 输入 | `变量 = 输入 提示` | `名字 = 输入 "请输入名字："` |
| 条件 | `如果 条件 就` | `如果 分数 >= 60 就` |
| 否则如果 | `否则如果 条件 就` | `否则如果 分数 >= 80 就` |
| 否则 | `不然` | `不然` |
| 当循环 | `当 条件 的时候` | `当 i < 10 的时候` |
| 固定次数循环 | `重复 次数 次 叫做 序号` | `重复 5 次 叫做 序号` |
| 遍历循环 | `遍历 集合 里的每一个 叫做 元素` | `遍历 列表 里的每一个 叫做 项` |
| 函数定义 | `功能 名称(参数)` | `功能 求和(a, b)` |
| 返回 | `返回 表达式` | `返回 a + b` |
| 异常处理 | `尝试 ... 如果出错 叫做 错误` | 见下方模板 |

## 2. 推荐模板

### 2.1 条件分支

```ym
如果 条件1 就
    显示 "命中条件1"
否则如果 条件2 就
    显示 "命中条件2"
不然
    显示 "兜底分支"
```

### 2.2 循环

```ym
当 计数 < 3 的时候
    显示 计数
    计数 += 1
```

```ym
重复 3 次 叫做 序号
    显示 序号
```

```ym
遍历 列表 里的每一个 叫做 当前
    显示 当前
```

### 2.3 函数

```ym
功能 加法(a, b)
    返回 a + b
```

### 2.4 异常处理

```ym
尝试
    内容 = 读文件("不存在.txt")
如果出错 叫做 错误
    显示 错误
```

### 2.5 模块

```ym
引入 "math" 叫做 数学库
显示 数学库.sqrt(144)

用 "math" 中的 pi
显示 pi
```

### 2.6 网格表格（GUI）

```ym
窗口 = 建窗口("表格示例", 600, 420)
表格 = 加表格(窗口, ["平台", "账号", "密码"], 10)
表格加行(表格, ["GitHub", "alice", "abc123"])
表格加行(表格, {"平台": "邮箱", "账号": "alice@qq.com", "密码": "******"})
打开界面(窗口)
```

### 2.7 模板字符串与字面量【】

```ym
名字 = "小明"
显示 "你好，【名字】"              # 模板替换
显示 "按钮：【【继续探索】】"      # 字面量【】
显示 "按钮：\【继续探索\】"        # 字面量【】
```

### 2.8 JSON + 本地数据库（SQLite）

```ym
引入 "数据工具" 叫做 数据
引入 "本地数据库" 叫做 库

配置 = {"作者": "景磊", "项目": "易码"}
数据.写JSON("配置.json", 配置)
显示 数据.读JSON("配置.json")

连接 = 库.打开数据库("演示.db")
库.执行SQL(连接, "CREATE TABLE IF NOT EXISTS 用户(id INTEGER PRIMARY KEY, 名字 TEXT, 年龄 INTEGER)")
库.执行SQL(连接, "INSERT INTO 用户(名字, 年龄) VALUES(?, ?)", ["小明", 18])
结果 = 库.查询SQL(连接, "SELECT 名字, 年龄 FROM 用户")
显示 结果
库.关闭数据库(连接)
```

### 2.9 路径与网络请求

```ym
引入 "系统工具" 叫做 系统
引入 "网络请求" 叫做 网络

目录 = 系统.拼路径(系统.当前目录(), "数据")
系统.创建目录(目录)
显示 系统.列出目录(系统.当前目录())

响应 = 网络.发GET("https://httpbin.org/get")
如果 响应["成功"] 就
    显示 "状态码：" + 响应["状态码"]
不然
    显示 "请求失败：" + 响应["错误"]
```

### 2.10 事务与 JSON 请求快捷接口

```ym
引入 "本地数据库" 叫做 库
引入 "网络请求" 叫做 网络

连接 = 库.打开数据库("订单.db", 错)  # 错=关闭自动提交，手动事务
尝试
    库.开始事务(连接)
    库.执行SQL(连接, "CREATE TABLE IF NOT EXISTS 订单(编号 TEXT, 金额 REAL)")
    库.执行SQL(连接, "INSERT INTO 订单(编号, 金额) VALUES(?, ?)", ["A001", 99.8])
    库.提交事务(连接)
如果出错 叫做 错误
    库.回滚事务(连接)
    显示 "事务失败：" + 错误
库.关闭数据库(连接)

接口结果 = 网络.发GET_JSON("https://httpbin.org/json")
如果 接口结果["成功"] 并且 接口结果["JSON"] != 空 就
    显示 "已解析 JSON"
```

## 3. 运算符（请用符号写）

- 算术：`+ - * / % ** //`
- 比较：`> < >= <= == !=`
- 逻辑：`并且`、`而且`、`或者`、`取反`、`!`

不推荐（当前不作为官方语法入口）：

- `加上` `减去` `乘以` `除以`
- `大于` `小于` `等于` `不等于` `至少是` `最多是`
