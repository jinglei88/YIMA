# 易码官方写法速查表

本页只放“推荐写法”。目标是：初学者照着写就能跑。

## 1. 先记住这 12 个句式

| 场景 | 官方推荐写法 | 示例 |
|---|---|---|
| 赋值 | `变量 = 表达式` | `名字 = "小明"` |
| 输出 | `显示 表达式` | `显示 名字` |
| 输入 | `变量 = 输入 提示` | `名字 = 输入 "请输入名字："` |
| 条件 | `如果 条件 就` | `如果 分数 >= 60 就` |
| 否则如果 | `否则如果 条件 就` | `否则如果 分数 >= 80 就` |
| 否则 | `不然` | `不然` |
| 当循环 | `当 条件 的时候` | `当 i < 10 的时候` |
| 固定次数循环 | `重复 次数 次 叫做 序号` | `重复 5 次 叫做 序号` |
| 遍历循环 | `遍历 集合 里的每一个 叫做 元素` | `遍历 列表 里的每一个 叫做 项` |
| 函数定义 | `功能 名称(参数)` | `功能 求和(a, b)` |
| 返回 | `返回 表达式` | `返回 a + b` |
| 异常处理 | `尝试 ... 如果出错 叫做 错误` | 见下方模板 |

## 2. 推荐模板

### 2.1 条件分支

```ym
如果 条件1 就
    显示 "命中条件1"
否则如果 条件2 就
    显示 "命中条件2"
不然
    显示 "兜底分支"
```

### 2.2 循环

```ym
当 计数 < 3 的时候
    显示 计数
    计数 += 1
```

```ym
重复 3 次 叫做 序号
    显示 序号
```

```ym
遍历 列表 里的每一个 叫做 当前
    显示 当前
```

### 2.3 函数

```ym
功能 加法(a, b)
    返回 a + b
```

### 2.4 异常处理

```ym
尝试
    内容 = 读文件("不存在.txt")
如果出错 叫做 错误
    显示 错误
```

### 2.5 模块

```ym
引入 "math" 叫做 数学库
显示 数学库.sqrt(144)

用 "math" 中的 pi
显示 pi
```

## 3. 运算符（请用符号写）

- 算术：`+ - * / % ** //`
- 比较：`> < >= <= == !=`
- 逻辑：`并且`、`而且`、`或者`、`取反`、`!`

不推荐（当前不作为官方语法入口）：

- `加上` `减去` `乘以` `除以`
- `大于` `小于` `等于` `不等于` `至少是` `最多是`
